# App Name: AppNexus
## App Description:
    'AppNexus' is a comprehensive App Directory platform designed to provide users with a seamless experience for discovering and managing applications. The platform features a dynamic Home page with curated top apps, an All Apps page equipped with real-time search and sorting by download counts, and a dedicated Installation Manager. Each app has a detailed profile showcasing technical specs, descriptions, and interactive review visualizations using data-driven charts. By leveraging local storage, the app allows users to simulate installations, track their personal app library, and manage them through a clean, intuitive interface. It bridges the gap between discovery and utility with a fully responsive design tailored for modern web users.

## Technologies Used:
    Core Framework & Routing:
    React.js: For building a component-based, high-performance User Interface.
    React Router Dom: To handle dynamic routing between Home, All Apps, Details, and Installation pages.
    Styling & UI Components:
    Tailwind CSS: For implementing the Figma design precisely with a utility-first approach.

    React Icons / Lucide: For consistent and scalable iconography across the navigation and buttons.

    Data Visualization & Logic
    Recharts: To render the responsive bar/pie charts for app ratings on the Details page.

    LocalStorage API: To persist "Installed" app data and maintain state even after page reloads.

    JSON Integration: For managing the structured database of 20 app objects.

    User Experience (UX)
    React-Toastify / React-Hot-Toast: To provide instant feedback via success/error toasts during installation.

    CSS Spinners: For smooth loading animations during search and page transitions.
